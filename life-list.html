Life List

Overview

With Life List app, we set out to make an easy and engaging bucket list app. The app will let you easily sign up and create bucket list items. If you feel you need to change them, we built in an Edit functionality. Then once youâ€™ve accomplished your goal, you can mark that item as complete! And if you want to reminisce on those old times, you can look at your Completed list.

The front end of the app was built using AngularJS and the Ionic Framework. This allowed for fully responsive design, as we built a web app, also, with the same functionality, that way you could use the app on any device. The backend was built using MongoDB as the database architecture running on MongoLab with RestifyJS as the API layer running on Node.js, leveraging Amazon Web Services for scalability and uptime. Since this was a hybrid app, we were able to use PhoneGap to package the app and submit to App Stores.

You can now download the Android and Apple iOS versions!
